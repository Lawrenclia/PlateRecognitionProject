## Project: SJTU ICE2607 License Plate Recognition System
### Description
This repository serves as the final project submission for the course ICE2607: Video Multimedia Retrieval Practice at Shanghai Jiao Tong University.

### Repository Structure

main branch: Contains the final, integrated codebase ready for deployment.
Member branches: Individual contributions and development history from each group member are preserved in their respective branches.

### Installation & Usage

Executable files (.exe) are available for download in the Releases section. You can run these directly to test the system.

### Note on Model Weights

Please note that the final model weights utilized in this project were obtained from pre-trained sources and were not trained from scratch by our team.
